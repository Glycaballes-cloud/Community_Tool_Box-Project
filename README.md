# Community_Tool_Box-Project
  **Overview**
  Community Tool Box is a community-focused application designed to help apartment residents share tools easily and efficiently. Instead of purchasing expensive tools that are rarely used, residents can discover, borrow, and return tools within their community through a centralized system with real-time availability.
  
  **Problem Analysis**
  In many apartment communities, residents independently purchase tools such as drills, ladders, or wrenches that are only used once or twice a year. There is no simple way to know which tools already exist nearby, resulting in unnecessary spending and duplicate ownership. Borrowing tools is often inconvenient or socially awkward, relying on door-to-door requests or informal group messages. Additionally, without tracking, communities lack insight into how often shared resources are actually used.
  
  **Solution**
  Community Tool Box solves this problem by providing a shared digital inventory of tools. Residents can see what tools are available, who owns them, and whether they are currently in use. The platform creates transparency, reduces friction in borrowing, and encourages sustainable sharing practices within the community.
  
  **Technical Approach**
  The application uses a form-based system to register tools along with owner details and location. Each tool has a status indicator (Available or Lent Out) that can be toggled by the owner. A searchable and filterable list allows users to quickly find tools by name or availability. Borrowing is handled through simple request interactions, and the system tracks usage data to calculate a utilization rate, measuring how effectively tools are shared.
