# Community_Tool_Box-Project
Community Tool Box is a shared-resource management application designed for apartment complexes and local communities to easily lend and borrow tools. Instead of buying expensive tools used only once or twice a year, residents can quickly find, request, and share tools with neighbors—no awkward door‑knocking required.

The platform allows residents to register tools, view real‑time availability, and request items directly from the owner. Tool owners can instantly toggle a tool’s status between Available and Lent Out, ensuring transparency and trust across the community.
